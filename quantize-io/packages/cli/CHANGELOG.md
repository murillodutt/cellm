# @quantize-io/cli — Changelog

## [0.1.0] — 2026-04-23

- `qt compress <path>` — JSON-first; exit 0/1/2.
- `qt benchmark <dir>` — gpt-tokenizer + Markdown/JSON table.
- `qt status` — mode + daemon presence + version + paths.
- `qt mode <level>` — writes flag via symlink-safe write.
- `qt rules [--level]` — filtered rules output.
- `qt uninstall` — strips hooks from `~/.claude/settings.json`; purges `~/.quantize/`.
