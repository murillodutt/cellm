# DSE Plugin

Design System Engine. Visual design skills for distinctive, production-grade frontend interfaces.

See [../README.md](../README.md) for full reference.

## Quick Start

```
/dse:frontend-ui "landing page for a fintech product"
```

## Structure

```
.claude-plugin/plugin.json   # Manifest
skills/                      # Design skills
  frontend-ui/SKILL.md       # Design thinking framework before code generation
```

## Philosophy

Skills inject a design thinking framework, not code tutorials. The Claude already knows how to code. What it needs is a decision framework: tone, typography, color, motion, composition — committed before the first line is written.
